2. "Trajectory Anomaly Detection based on Similarity Analysis"

Autores:

  1. Harley Vera-Olivera
  2. Lauro Enciso-Rodas
Descripcion:

El estudio presenta un análisis comparativo del rendimiento de dos descriptores de trayectorias para detectar anomalías. Se utilizan las transformadas de ondícula y Fourier para generar características y luego identificar anomalías mediante aprendizaje no supervisado. Los resultados demuestran la viabilidad de utilizar descriptores de trayectorias junto con técnicas de clustering para la detección automática de anomalías.

3. "Clustering Analysis for Traffic Jam Detection for Intelligent Transportation System"

Autores:

  1. Harley Vera-Olivera
  2. José Luis Soncco-Álvarez
  3. Edwin Alvarez-Mamani
Descripcion:

La investigación emplea el algoritmo de Optimización Paralela de Arañas Sociales (P-SSO) para detectar atascos de tráfico en Cusco, Perú. Se basa en datos geoespaciales de autobuses de transporte público entre junio y agosto de 2019. Se desarrolló una aplicación para visualizar gráficamente los clusters obtenidos.
